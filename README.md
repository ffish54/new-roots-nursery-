# New Roots Nursery Website

## Project Overview
this project is an advertisement  website for my family's small plant nursery. the projects goal is to create a website that can advertise and provide information about the nursery and plants sold to potential customers in the local area. 


smaller goals based on user storys to create the website so that it fully fulfills its purpose 

User Story:
As a potential customer, I want to browse available plants so that I can see what the nursery offers before visiting. - Must have

Acceptance Criteria:
The website includes a section displaying all available plants.
Each plant listing includes a name, image, price and short description.
The plant listings are displayed clearly and are easy to browse on both desktop and mobile devices.

Tasks:
Add a plant list on a separate “Plants” page.
Add plant entries with images, names, prices and descriptions.
Ensure the layout is responsive.


User Story:
As a potential customer, I want to contact the nursery so that I can ask about plant availability or care advice. - Must Have

Acceptance Criteria:
A contact section that includes a phone number and email address.
Contact information is clearly labelled and easily findable.
The section is accessible from the main navigation.

Tasks:
Create a contact page.
Add phone number and email address.
Link the page in the navigation menu.

User Story: 
as a potential customer I want the website to be easily navigable so i dont get lost exploreing the site. - Should have

Acceptance Criteria:
a clear navigation section that has links to each page 
navigation available on every page 
navigation should be usable on all screen sizes and mobile phones

Tasks:
create a navigation section that links to every page 
have a navigation section on everypage 
have the navigation section fit every screen size
